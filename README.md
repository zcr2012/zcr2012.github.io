# 心流博客 - 个人博客系统

一个现代化、功能完善的个人博客系统，支持文章管理、用户系统、评论功能等。

![博客截图](screenshot.png)

## ✨ 功能特性

### 📝 文章管理
- **文章发布**：支持富文本编辑器，可格式化文本
- **文章分类**：支持技术、生活、思考、其他等分类
- **文章搜索**：全文搜索功能，快速查找文章
- **文章统计**：阅读量统计，实时更新

### 👥 用户系统
- **用户注册**：支持用户注册和登录
- **角色管理**：普通用户和管理员角色
- **账户安全**：密码加密存储，登录尝试限制
- **用户管理**：管理员可管理所有用户（查看、编辑、锁定、删除）

### 💬 评论系统
- **评论发表**：支持注册用户和访客发表评论
- **评论管理**：管理员可置顶、删除评论
- **实时验证**：表单验证和字符计数
- **草稿保存**：自动保存评论草稿

### 🎨 UI设计
- **现代化界面**：渐变色彩、圆角设计、阴影效果
- **响应式布局**：完美适配手机、平板、桌面
- **全屏阅读**：文章阅读全屏体验
- **暖色调主题**：友好的视觉体验

### 🔧 管理后台
- **仪表盘**：实时统计文章、用户、评论数据
- **文章管理**：编辑、删除、发布文章
- **用户管理**：查看、编辑、锁定、删除用户
- **评论管理**：审核、置顶、删除评论
- **数据导出**：导出所有数据备份

## 🚀 快速开始

### 安装步骤

1. **克隆项目**
   ```bash
   git clone https://github.com/yourusername/blog.git
   cd blog
   ```

2. **启动应用**
   - 方式1：使用Python内置服务器
     ```bash
     python -m http.server 8000
     ```
   - 方式2：使用Node.js http-server
     ```bash
     npx http-server -p 8000
     ```
   - 方式3：直接打开index.html文件

3. **访问应用**
   打开浏览器访问：`http://localhost:8000`

## 📖 使用指南

### 作为管理员

1. **登录**：使用管理员账户登录
2. **发布文章**：点击"写文章"按钮
3. **管理用户**：进入后台管理 → 用户管理
4. **管理评论**：进入后台管理 → 评论管理
5. **查看统计**：在后台仪表盘查看实时数据

### 作为普通用户

1. **注册账户**：点击登录页面的"注册"链接
2. **发表评论**：在文章页面点击"写评论"
3. **个人中心**：查看"我的文章"（如果有发布权限）

### 作为访客

1. **浏览文章**：查看所有公开文章
2. **发表评论**：可以填写昵称发表评论
3. **搜索文章**：使用搜索功能查找感兴趣的内容
